## Introduction

A butterfly has two wings that are mirror images of each other. A snowflake has six arms, each identical, rotated around a centre point. A tiled floor repeats the same shape in every direction. All of these are patterns with precise rules. A computer can generate any of these patterns if you give it the right algorithm. The algorithm for a symmetrical pattern always involves a loop, because symmetry means doing the same thing multiple times in a consistent way. Today you are going to find the rule behind a symmetrical pattern and write the algorithm that produces it.

## What We Will Learn

- What line symmetry is and how to check whether a shape has it.
- How a symmetry rule can be written as an algorithm.
- How loops are used to generate patterns with multiple lines of symmetry.
- How to design a symmetrical pattern and write the algorithm behind it.

## Core Explanation

**Line symmetry**

A shape has line symmetry if you can fold it along a straight line and both halves match exactly. The fold line is called the **line of symmetry** or **mirror line**. A square has four lines of symmetry. A rectangle has two. A regular hexagon has six. An irregular shape may have none.

To check for line symmetry, place a mirror along the proposed fold line. If the reflection completes the shape, the line is a line of symmetry.

**Symmetry as an algorithm**

A symmetrical pattern is generated by a rule. For a shape with one line of symmetry, the rule is: draw one half, then reflect it across the line. For a shape with rotational symmetry, such as a snowflake, the rule is: draw one arm, rotate by a fixed angle, draw another arm, repeat until you return to the start.

This is a loop. The snowflake algorithm looks like this.

```
draw one arm
rotate 60 degrees
repeat 5 more times:
  draw the same arm
  rotate 60 degrees
```

Six arms, each rotated 60 degrees from the previous, produce a shape with six-fold rotational symmetry. The loop makes the algorithm compact: you describe one arm and then repeat.

**Tile symmetry**

A repeating tile pattern has translation symmetry: the same tile shifted left or right or up or down produces the whole pattern. The algorithm is a nested loop.

```
repeat for each row:
  repeat for each column in that row:
    draw the tile
```

## Activity

**Symmetry Hunt and Algorithm Design** (33 minutes)

Part 1: Symmetry hunt (8 minutes)

Step 1 (4 minutes): Display six shapes on the board: a square, an equilateral triangle, the letter A, the letter S, a regular pentagon, a right-angled scalene triangle. For each shape, ask: "Does it have line symmetry? How many lines?" Students write their answers.

Correct answers: square (4), equilateral triangle (3), A (1), S (0), pentagon (5), scalene triangle (0).

Step 2 (4 minutes): Ask two students to demonstrate with a mirror card. Place the mirror along a proposed line of symmetry. Does the reflection complete the shape? If yes, it is a line of symmetry.

Ask: "What do you notice about the relationship between the number of sides and the number of lines of symmetry for regular shapes?" (For a regular polygon with n sides, there are n lines of symmetry.)

Part 2: Writing symmetry algorithms (10 minutes)

Step 3 (5 minutes): Together, write the algorithm for drawing the letter A on squared paper. The A has one vertical line of symmetry. The algorithm:

```
draw the right half of the letter A
reflect across the vertical centre line
```

Simplified to actual steps: "Draw a diagonal line up-right to the top centre. Draw a horizontal line one square right from the centre. The left half mirrors the right half exactly."

Ask: "If this were a loop, what would repeat?" (The drawing action, once for each half.)

Step 4 (5 minutes): Each student writes an algorithm for drawing a shape with two lines of symmetry using squared paper. They choose a rectangle or a diamond. The algorithm must describe one quarter, then state how the other three quarters are generated by reflecting.

Part 3: Design and draw (15 minutes)

Step 5 (5 minutes): Each student designs a tile for a symmetrical floor pattern. The tile must fit in a 4x4 square and have at least one line of symmetry. They colour it and check with a mirror.

Step 6 (5 minutes): They write the algorithm for their tile. It must name the line of symmetry, describe how to draw one half, and state the reflection rule.

Step 7 (5 minutes): They extend the tile to fill a larger area by repeating it in a 3x3 grid. They write two lines describing this as a nested loop: "For each row, for each column, draw the tile."

## Real-World Connection

Architects use symmetry algorithms when designing buildings. A symmetric facade is specified once for one half, and software mirrors it to produce the other half. Logo designers use rotation algorithms: draw one element of a logo, rotate by a fixed angle, repeat until the full circle is complete. This is exactly the snowflake algorithm. One description, many applications.

## Check for Understanding

1. A regular hexagon has six lines of symmetry. Write a loop-based algorithm that would draw it by repeating one arm six times. How many degrees does each rotation need?

2. A pattern tile has two lines of symmetry. If you describe one quarter of the tile, how many reflect or rotate operations are needed to complete the other three quarters?

3. Why is a loop the natural structure for an algorithm that generates a pattern with rotational symmetry?

## Extension and Differentiation

**For students who need more support:** Use a pre-drawn shape with one line of symmetry marked. Students trace the shape, fold along the line, and check the two halves match. Then write two sentences: "One half is [description]. The other half is the mirror image."

**For students who want a challenge:** Design a tile pattern that has both line symmetry and rotational symmetry. Write an algorithm that uses a nested loop to tile a 4x4 grid with your design. Count how many separate draw operations the algorithm performs compared to drawing each tile manually.

## Summary

- A shape has line symmetry if folding along the mirror line produces two identical halves.
- Symmetrical patterns are generated by algorithms: draw one unit, reflect or rotate, repeat.
- The repeat step in a symmetry algorithm is a loop. The number of iterations matches the number of lines or angles of symmetry.
- Nested loops generate tiled patterns by repeating the tile across rows and columns.
